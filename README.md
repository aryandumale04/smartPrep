The **AI-Powered Interview Preparation App** is a full-stack MERN application designed to help users prepare for job interviews through intelligent, role-based, and interactive Q&A sessions.  
The system generates context-aware interview questions and answers using **Google Gemini API** and provides users with detailed insights on their performance.
